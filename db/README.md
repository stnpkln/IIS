# IIS - REST API

IIS project - Social Network

## Dependencies

docker

## Start DB

- start docker
- in /db run `docker-compose up`

## Connect to DB

- the DB can be accessed on `localhost:5432`

## Adminer

- the DB adminer runs on `localhost:8080`
- type: PostgreSQL
- host: iisDb
